Block UI service - AngularJS reusable UI component

Description

Lightweight AngularJS service which implements the block UI functionality. This component depends on only the AngularJS.

  1. Configurable message
  2. Configurable blocked UI time (optional)
  3. CSS3 animation spinner

Click the button below to block the UI (it will automatically enable the UI after 3 seconds).
